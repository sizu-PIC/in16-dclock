# in16-dclock

このリポジトリは IN-16 ニキシー管を用いたニキシー管時計の C ソースコードを管理するリポジトリです．

気が向いたら Qiita や elchika で記事を書きたいと思います．

## 環境

| 項目 | 使用した物 |
|:---:|:---:|
| マイコン | PIC16F1829 |
| IDE | MPLABX |

MPLABX のバージョンとかもちゃんと調べておきたいですね．
